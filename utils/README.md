# Utilities

General-purpose utility scripts and helpers.

## Planned Utilities

- `download_models.py` — Script to download all pre-trained checkpoints
- `download_data.py` — Script to download all datasets
- `visualize_predictions.py` — Visualize model predictions alongside input images
- `dataset_stats.py` — Compute dataset statistics (number of samples, text length distribution)
- `convert_hf_to_fairseq.py` — Convert HuggingFace models to fairseq format (and vice versa)

## Key Utility from Source

- `src/convert_to_SROIE_format.py` — Converts fairseq generation output into the SROIE submission zip format
