# Notebooks

Jupyter notebooks for experimentation and demos.

## Planned Notebooks

- `inference_demo.ipynb` — Step-by-step inference demonstration with a TrOCR model
  - Load a pre-trained model
  - Process a sample text-line image
  - Generate and display the predicted text
  - Compare with ground truth

## Running Notebooks

```bash
pip install jupyter
jupyter notebook notebooks/
```
